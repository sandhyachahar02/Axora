# 🚀 Axora – AI-Powered Team Building Platform

Axora is a modern AI-driven platform designed to help students, developers, and innovators **find the perfect team, collaborate efficiently, and build impactful projects** — especially for hackathons and collaborative environments.

---

## 🌟 Problem Statement

Finding the right teammates is one of the biggest challenges in hackathons and collaborative projects. Most platforms lack:

* Smart team matching
* Skill-based collaboration
* Organized project workflows

---

## 💡 Solution

Axora solves this by combining **AI-powered matching + collaboration tools** in a single platform.

👉 It helps users:

* Discover like-minded people
* Build balanced teams
* Manage projects efficiently

## 🚀 Getting Started

### 🔹 Clone the repo

```bash
git clone https://github.com/your-username/Axora.git
cd Axora
```

---

### 🔹 Setup Backend

```bash
cd axora-api
pip install -r requirements.txt
uvicorn main:app --reload
```

---

### 🔹 Setup Frontend

```bash
cd axora-next
npm install
npm run dev
```
